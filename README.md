## Sobre o funcionamento
A pagina inicial contem um formulario para cadastrar a barbearia, apos clicar no botão de cadastro, o backend ira validar as informações e em caso de credencias incorretas enviara uma mensagem para o frontend com os detalhes. Também consta com um formulario para login caso já tenha cadastrado, ao entrar na conta, possui a opção de sair da conta, e remover o cadastro, aonde o mesmo ira remover as informações sobre o mesmo no banco de dados.

## Nota sobre a Hospedagem

Como a API deste projeto está hospedada no plano gratuito do Render, o servidor pode entrar em estado de repouso após alguns minutos de inatividade. Por conta disso, a primeira requisição (como o primeiro envio de formulário) pode demorar cerca de 50 segundos para responder enquanto o servidor acorda. Após esse carregamento inicial, as respostas seguintes serão instantâneas.
